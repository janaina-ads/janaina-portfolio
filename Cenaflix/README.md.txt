# 🎬 CENAFLIX - Sistema de Cadastro de Filmes

## 📌 Descrição
Aplicação desktop desenvolvida em Java para cadastro e gerenciamento de filmes.

## 🚀 Funcionalidades
- Cadastro de filmes
- Listagem em tabela (JTable)
- Exclusão de registros
- Busca dinâmica por categoria

## 🛠️ Tecnologias utilizadas
- Java
- Swing
- MySQL
- JDBC
- NetBeans
